# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in BotTimer, **please do not open a public GitHub Issue**.

Report it privately by email:

**akimasa.dev.ai@gmail.com**

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

As this is an individual project, a response may take a few days. I will do my best to address confirmed vulnerabilities promptly and keep you informed of progress.

## Scope

BotTimer communicates only with the official SwitchBot API (`api.switch-bot.com`) over HTTPS.  
It does not collect personal data, use analytics SDKs, or connect to any other servers.

## Non-Security Support

For general questions, bug reports, and feature requests, please use:

- [GitHub Issues](https://github.com/nnakimasa/BotTimer/issues)
- [Support page](https://nnakimasa.github.io/bottimer-privacy/support.html)
