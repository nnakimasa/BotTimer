BotTimer

SwitchBot デバイスをタイマー制御する非公式アプリ

Android 版ダウンロード

APK をダウンロード BotTimer-v1.0.0.apk (69.6 MB)

バージョン 1.0.0 ・ Android 5.0 以上対応

インストール手順

  1. 上のボタンから APK ファイルをダウンロードします。
  2. Android の 設定 → セキュリティ（またはアプリ） を開き、「提供元不明のアプリ」 または 「不明なアプリのインストール」 を許可します。
    （端末・Android バージョンにより場所が異なります）
  3. ダウンロードした APK ファイルをタップしてインストールします。
  4. インストール完了後、アプリを起動します。
⚠️ このアプリは Google Play Store 経由ではなく直接配布しているため、 インストール時に Android から警告が表示されることがあります。 これは正常な動作です。安全を確認した上でインストールしてください。

アプリについて

BotTimer は SwitchBot 製品（プラグミニ等）をタイマーで ON/OFF 制御できる非公式コンパニオンアプリです。 SwitchBot 公式 API のみを使用しており、SwitchBot Inc. とは無関係です。

利用には SwitchBot アカウントの API トークンが必要です。 SwitchBot アプリの設定画面から取得できます。
