プライバシーポリシー

対象アプリ: BotTimer / 最終更新日: 2026年5月16日

本プライバシーポリシーは、BotTimer（以下「本アプリ」）における利用者情報の取扱いについて定めるものです。本アプリは利用者の個人情報を収集・送信しないことを基本方針とし、必要最小限の情報のみを利用者端末内で保管します。

1. 取得する情報

本アプリは、以下の情報のみを利用者の端末内に保存します。これらの情報は当開発者を含むいかなる第三者にも送信されません。

  • SwitchBot API トークン・シークレット・デバイス ID（利用者ご自身で入力した値）
  • タイマー設定値（初期時間、警告時間、ロックモード、インターバル設定など）
  • 進行中のタイマーの終了予定時刻（アプリ再起動後のタイマー復元のため）

2. 情報の利用目的

取得した情報は、本アプリのタイマー機能および SwitchBot 機器の制御を実現する目的でのみ使用します。広告配信、利用分析、その他の目的では一切使用しません。

3. 通信先となる外部サービス

本アプリは以下の外部サービスとのみ通信します。いずれも利用者自身の操作に基づくものであり、当開発者が運営するサーバーは存在しません。

  • SwitchBot API（api.switch-bot.com）
    利用者が入力した認証情報を用いて、SwitchBot 機器の操作・一覧取得を行います。送信されるのはご自身の認証情報と機器操作コマンドのみで、当開発者は通信内容を傍受・保管しません。
    SwitchBot 社のプライバシーポリシーは こちら をご参照ください。

4. プッシュ通知について

タイマー終了をお知らせするために、iOS/Android のローカル通知機能を使用します。通知は利用者の端末内で完結し、外部サーバーから送信されるものではありません。

5. 解析ツール・トラッキング

本アプリは Google Analytics、Firebase Analytics、広告 SDK、クラッシュレポートツール等の解析ツールを一切使用していません。利用者の行動を追跡することはありません。

6. 第三者への情報提供

本アプリは利用者の情報を第三者に提供することはありません。前述の SwitchBot API への通信は、利用者ご自身の指示に基づき行われるものです。

7. 情報の管理

本アプリで保存される情報は、利用者の端末内のローカルストレージ（iOS: UserDefaults、Android: SharedPreferences）に保管されます。当開発者は当該情報にアクセスする手段を持ちません。

8. 利用者の権利

  • 設定画面から各種設定値を編集・削除できます。
  • アプリをアンインストールすることで、本アプリが端末に保存したすべての情報が削除されます。

9. 子どものプライバシー

本アプリは 13 歳未満の子どもを対象としていません。13 歳未満の子どもから意図的に情報を収集することはありません。

10. プライバシーポリシーの変更

本ポリシーは予告なく変更されることがあります。変更があった場合は本ページ上で告知し、最終更新日を更新します。

11. SwitchBot との関係

本アプリは SwitchBot 社の公式アプリではなく、同社が提供する公開 API を用いた個人開発の非公式アプリです。SwitchBot は SwitchBot 社の商標です。
